## Priorités de ciblage (tableau de synthèse)

### Mois 1-2 (actions rapides, KD < 20)
Mots-clés IA + gratuit : `cv ia`, `cv ia gratuit`, `créer cv avec chatgpt`, `générateur lettre motivation ia`
**Pourquoi** : Faible concurrence, intention forte, alignés sur la proposition de valeur

### Mois 2-4 (pages modèles)
Cluster 2 complet + premiers articles blog sur l'ATS
**Pourquoi** : Volume élevé, convertissent directement

### Mois 4-6 (contenu blog)
Articles longs sur les mots-clés informationnels + pages alternatives concurrents
**Pourquoi** : Construire l'autorité domaine progressivement

### Mois 6-12 (SEO programmatique)
Pages `/modeles-cv/[metier]` générées automatiquement (100+ pages)
**Pourquoi** : Effet de levier exponentiel sur la longue traîne

## Notes pour la mise à jour

Ce tableau est un point de départ basé sur des estimations. À valider avec :
- **Semrush** ou **Ahrefs** pour les volumes exacts
- **Google Search Console** (une fois le site indexé) pour les impressions réelles
- **Google Keyword Planner** pour les données CPC (utile pour le SEA aussi)
